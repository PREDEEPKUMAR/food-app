# FoodieSpot - Food Ordering App

A modern React food ordering application featuring **Veg** and **Non-Veg** sections with 3 premium outlets.

## Features

- **Veg & Non-Veg Sections** — Filter dishes by dietary preference
- **3 Outlets** — Spice Garden (Bangalore), Royal Kitchen (Delhi), Coastal Flavors (Mumbai)
- **Shopping Cart** — Add/remove items with quantity controls
- **Responsive Design** — Works on desktop, tablet, and mobile
- **Order Placement** — Place orders with success confirmation

## Outlets

| Outlet | Location | Rating | Delivery |
|--------|----------|--------|----------|
| Spice Garden | MG Road, Bangalore | 4.5 | 25-35 min |
| Royal Kitchen | Connaught Place, Delhi | 4.7 | 30-40 min |
| Coastal Flavors | Marine Drive, Mumbai | 4.3 | 20-30 min |

## Tech Stack

- React 19 + Vite
- CSS3 (no external UI libraries)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
